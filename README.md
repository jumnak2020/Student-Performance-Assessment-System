# Student-Performance-Assessment-System
## Overview
The Student Performance Assessment System is a web-based application designed to manage and analyze academic performance for both school and college students. The system provides institution-specific access, performance evaluation, and machine learning-based prediction of student outcomes.

## Features
- Role-based access for Students, Teachers, School Admins, and College Admins
- Institution-specific data management
- Interactive performance visualization using Plotly
- Performance classification (Fast / Average / Slow Learner)
- Machine learning prediction using Linear Regression, Random Forest, and SVM
- Automated PDF report generation

## Technologies Used
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Machine Learning: scikit-learn
- Visualization: Plotly
- Report Generation: ReportLab
- Version Control: Git & GitHub
